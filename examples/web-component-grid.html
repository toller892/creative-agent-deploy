ADCP Creative Grid

Using Web Components for easy embedding

How to Use

<!-- 1. Include the web component script -->
<script src="https://creative.adcontextprotocol.org/static/rendered-creative.js"></script>

<!-- 2. Use the component anywhere -->
<div class="grid">
    <rendered-creative
        src="https://preview-url.com/uuid/desktop.html"
        width="300"
        height="400">
    </rendered-creative>
</div>
